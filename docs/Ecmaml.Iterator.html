Module Ecmaml.Iterator

module Iterator: sig .. end

Implementation of an iterator
type 'a it_result =
| Done (*
Done is the representation of the javascript object { done: true }. It's used when the iterator does not return any value and just want to mark the end.
*)
| Return of 'a (*
Return elt is the representation of the javascript object { done: true, value: elt }. It's used when the iterator has finished and want to return a value.
*)
| Yield of 'a (*
Yield elt is the representation of the javascript object { done: false, value: elt }.
*)
The Type of the result of the iterator next function.
val identity : 'a -> 'b
identity elt returns the same ocaml value to an another type. It's usefull when translating the javascript value into the ocaml value when they share the same representation. For example, this function can be use to translates javascript integers into ocaml int.
